# Blinkit Sales & Operations Dashboard (Tableau)

## Project Overview
This project focuses on analyzing Blinkit's sales and operational performance using Tableau.  
The goal of this dashboard is to explore how different product categories, outlet sizes, and item characteristics influence overall sales performance.

Instead of only looking at total revenue, this dashboard breaks down the data to reveal meaningful insights that help understand customer purchasing patterns and product performance.

---

## Tools Used
- Tableau
- CSV Dataset

---

## Dashboard Highlights

### Key Performance Indicators
The dashboard includes important KPIs such as:
- Total Sales
- Average Item Visibility
- Item Count
- Average Sales per Item

These metrics provide a quick overview of overall performance.

### Outlet Size Contribution
A treemap visualization shows how sales are distributed across different outlet sizes (Small, Medium, and High).

### Item Visibility vs Sales
This visualization explores how product visibility impacts sales across different item categories.

### Product Category Performance
A category-level analysis highlights which item types generate the highest sales.

### Fat Content Analysis
The dashboard compares sales between Low Fat, Regular, and Non-Edible product types.

---

## Key Insights
- Small outlets contribute the highest share of total sales.
- Certain product categories consistently outperform others.
- Higher item visibility generally correlates with stronger sales.
- Low-fat products show strong demand compared to regular items.
